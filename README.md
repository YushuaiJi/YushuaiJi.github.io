# Yushuai Ji's Academic Homepage

This repository hosts the personal academic homepage of Yushuai Ji at:

https://yushuaiji.github.io

The site is built with Jekyll and published through GitHub Pages. The main editable pages are:

- `_pages/about.md`
- `_pages/publications.html`
- `_pages/cv.md`
- `assets/css/academic.css`

To preview locally:

```bash
bundle install
bundle exec jekyll serve
```
